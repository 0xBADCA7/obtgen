# OBTGEN
Create your own openbox-theme from .Xresources

## Usage
```sh
 $ git clone https://github.com/fikriomar16/obtgen.git
 $ cd obtgen/
 $ ./obtgen
   or
 $ ./obtgen --wal #Generate theme from pywal cache
```
Then open `lxappearance`, and select "myobtheme"
